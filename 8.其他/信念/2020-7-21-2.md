# 一切从实际出发
- 想问题办事情要把客观存在的事物作为根本出发点
- 客观分析认识事物，从它的属性、行为、状况出发。了解它、认识它。

1. 区分事物
```
是什么？
有什么用？能解决什么问题？有什么价值？
由什么构成？
现在的状况是什么？
```
2. 联系地发展地看问题
```
它存在的条件是什么？
它和周围其他事物有怎样的联系？它影响着什么？它受谁制约？
它的发展历史是什么？
它又将如果发展下去?
```
